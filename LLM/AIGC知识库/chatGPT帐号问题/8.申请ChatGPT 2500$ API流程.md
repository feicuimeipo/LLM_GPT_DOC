#### 2、填表申请

填表最重要三点

- 1、名字，我写的是英文名，姓氏写的自己的。

- 2、居住地选择OpenAI支持的地方，不要写中国，我选择的是新加坡

- 3、业务描述，直接ChatGPT生成。但是有几个重点。

  

提示语模版下:

```
我们是[公司或者个人]

[主要做什么]

想申请[用来做什么]

帮我用英文写一份申请书”

请语气要诚恳，表达我迫切的愿望

列出中文版，然后再翻译成英文。

如果你有更好的建议，请帮忙补充

我写的提示词
```
