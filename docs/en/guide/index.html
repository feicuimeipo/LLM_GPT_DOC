    # 大屏设计软件

    DataCanvas是一款功能强大的大屏工具软件，可以实现大屏设计与预览，以SpringBoot+ElementUI+Vue等技术为架构，同时支持MySQL、Oracle等数据集接入，使用简单。

    # 功能介绍

    # 大屏管理

    支持大屏增、删、改、设计、预览与复制等操作

    # 设计器

    以拖拉拽的方式进行大屏的可视化设计，内置有各种图表、边框及修改组件

    # 资源库

    除了内置了资源库外，还地持自定义上传。这些资源可直接在大屏设计器中引用，资源图像是指: 3D边框图片、装饰条、背景图

    # 数据源

    支持MySQL、Oracle、PostgreSQL、ClickHouse数据库接入

    # 数据集

    支持各种格式的数据集：包含：JSON数据集、脚本数据集、JS脚本数据集、HTTP数据集接入等多种方式.
